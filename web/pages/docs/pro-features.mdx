1. Go to **Dashboard** ‚Üí **Policies**

```yaml
# .klyntos/policies/payment-security.yml
name: Payment Security Policy

severity: critical

rules:
  - id: no-card-logging
    pattern: ".*log.*(card|cvv|credit).*"
    message: "Never log credit card information"

  - id: encrypted-storage
    pattern: ".*save.*(card|payment).*"
    requires: ["encryption", "tokenization"]

```

Apply it:

```bash
kg scan checkout.py --policy payment-security
```

### API Access ‚úÖ LIVE

Integrate KlyntosGuard into your CI/CD pipeline via REST API.

#### GitHub Actions

```yaml
# .github/workflows/security.yml
name: Security Scan

on: [push, pull_request]

jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2

      - name: KlyntosGuard Scan
        uses: klyntos/guard-action@v1
        with:
          api-key: ${{ secrets.KLYNTOS_GUARD_API_KEY }}
          fail-on: critical,high
          output: sarif

      - name: Upload Results
        uses: github/codeql-action/upload-sarif@v2
        with:
          sarif_file: klyntos-guard-results.sarif
```

## API Reference

### Scan Endpoint ‚úÖ LIVE

The main scanning endpoint with tier-based features:

**Endpoint:** `POST /api/v1/scan`

**Headers:**
```
Authorization: Bearer <jwt_token>
Content-Type: application/json
```

**Request Body:**
```json
{
  "code": "your code here",
  "language": "python",
  "filename": "app.py",
  "policies": ["all"],
  "model": "opus",        // Pro only: "haiku" or "opus"
  "depth": "deep"         // Pro only: "standard" or "deep"
}
```

**Basic Plan Restrictions:**
- Maximum 1,000 scans/month (enforced)
- Always uses `model: "haiku"`
- Always uses `depth: "standard"`
- Returns 429 error when limit exceeded

**Pro Plan Benefits:**
- Unlimited scans
- Can specify `model: "opus"` for advanced analysis
- Can specify `depth: "deep"` for comprehensive review
- 8192 token limit (vs 4096 for Basic)

**Response:**
```json
{
  "success": true,
  "result": {
    "violations": [
      {
        "line": 10,
        "severity": "critical",
        "category": "hardcoded-secret",
        "message": "Hardcoded API key detected",
        "suggestion": "Move API key to environment variables",
        "code_snippet": "api_key = 'sk-1234...'",
        "dataflow": "User input (line 5) ‚Üí Database (line 10)"  // Deep mode only
      }
    ],
    "summary": {
      "critical": 1,
      "high": 2,
      "medium": 3,
      "low": 1,
      "info": 0
    },
    "scan_time_ms": 5234
  },
  "metadata": {
    "language": "python",
    "filename": "app.py",
    "policies": ["all"],
    "lines_of_code": 150,
    "model_used": "opus",      // Reflects actual model used
    "analysis_depth": "deep",  // Reflects actual depth used
    "plan_tier": "pro"         // Shows user's plan
  }
}
```

**Error Response (Limit Exceeded):**
```json
{
  "error": "Monthly scan limit reached",
  "message": "You have reached the 1,000 scans/month limit for Basic plan. Upgrade to Pro for unlimited scans.",
  "current_usage": 1000,
  "limit": 1000,
  "upgrade_url": "/pricing"
}
```
